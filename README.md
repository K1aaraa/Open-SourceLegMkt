# Open-SourceLegMkt

Internal, authenticated Marketing & Community Operations Hub for the Open-Source Leg (OSL), including asset library, task tracking, and an internal-only AI assistant grounded in approved docs.

## Local development

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
